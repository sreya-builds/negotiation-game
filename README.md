# 🧠 Negotix — AI Negotiation Game

Negotix is an interactive AI-powered negotiation game where users bargain with a simulated seller to get the lowest possible price.
The game is not just about entering numbers — it’s about strategy, timing, and understanding behavior.

 🚀 Live Demo
👉 https://your-vercel-link.vercel.app  
👉 Backend API: https://your-render-link.onrender.com

🎯 Project Goal
To simulate real-world negotiation scenarios using:
- Behavior-based AI responses
- Hidden constraints (minimum price)
- Multi-round decision making
- Competitive leaderboard

🧩 Features
🤖 Smart AI Seller
- Different personalities:
  - Friendly
  - Aggressive
  - Smart
  - Pressure-based
🎯 Real Negotiation System
- Hidden minimum price
- Dynamic counter offers
- Strategy-based responses
🧠 Behavior Detection
AI detects user intent:
- Student appeal
- Polite tone
- Urgency
- Budget constraints
- Walk-away tactics
🔄 Multi-Round Gameplay
- Limited rounds (max 5)
- Each move affects outcome
- 🏆 Leaderboard
- Lowest price wins
- Fewer rounds = higher rank
🎮 Replayable Game
- Random product every time
- Random seller personality

🏗️ Tech Stack

### Frontend

### Backend
- Node.js
- Express.js
